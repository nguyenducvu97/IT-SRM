# Logo Installation Instructions

## Logo Placement

The logo "VuIT_SGI VINA" has been configured to appear in the center of the header.

## File Location

Please place your logo file at:
```
c:\xampp\htdocs\it-service-request\assets\images\logo-vuit-sgi-vina.png
```

## Logo Specifications

- **Recommended size:** 200x50 pixels (width x height)
- **Format:** PNG with transparent background
- **Maximum dimensions:** 200px width, 50px height
- **File name:** logo-vuit-sgi-vina.png

## What has been implemented:

1. **HTML Structure:** Added header-center div with logo image
2. **CSS Layout:** 
   - 3-column header layout (left, center, right)
   - Logo centered in the middle
   - Responsive design for mobile and tablet
3. **Responsive Behavior:**
   - Desktop: Logo centered between title and user info
   - Tablet: Smaller logo (180x45px)
   - Mobile: Logo on top, stacked layout (150x40px)

## Header Layout:
```
[IT Service Request]    [VuIT_SGI VINA Logo]    [User Info Logout]
```

## Mobile Layout:
```
     [VuIT_SGI VINA Logo]
[IT Service Request]
    [User Info Logout]
```

## After placing the logo:
1. Copy your logo file to the specified location
2. Refresh the page
3. The logo should appear centered in the header

## If logo doesn't appear:
1. Check file path and name
2. Verify file permissions
3. Check browser console for 404 errors
4. Ensure the file is a valid PNG image

The layout is fully responsive and will adapt to different screen sizes automatically.
