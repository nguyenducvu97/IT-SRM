🔍 BROWSER DEBUG CHECKLIST

Vấn đề: Staff nhận yêu cầu nhưng admin và user không nhận được thông báo

Nguyên nhân: Browser session/authentication issue

📋 STEP-BY-STEP DEBUG

1️⃣ Clear Browser Cache

Nhấn Ctrl + Shift + Delete và chọn "All time"

2️⃣ Login as Staff

Đi đến trang login và nhập:

Username: staff1
Password: password123

3️⃣ Check Session

Kiểm tra session có hoạt động không

4️⃣ Test API Call

Test API call trực tiếp

5️⃣ Check Console Errors

Mở F12 và kiểm tra console có lỗi không

6️⃣ Test Accept Request

Đi đến trang chi tiết yêu cầu và click "Nhận yêu cầu"

🔧 TROUBLESHOOTING

❌ Common Issues

  • Session expired: Login lại
  • Wrong credentials: Đảm bảo dùng staff1/password123
  • Cache issues: Clear cache hoàn toàn
  • Network errors: Kiểm tra internet connection
  • JavaScript errors: Kiểm tra console F12

✅ Expected Results

  • Login thành công và thấy dashboard
  • Có thể xem danh sách yêu cầu
  • Click "Nhận yêu cầu" không có lỗi
  • Request status chuyển thành "in_progress"
  • Admin và user nhận được thông báo

📞 Contact Support

Nếu vẫn không hoạt động, cung cấp:

  • Screenshot của console errors
  • Screenshot của Network tab
  • Browser version và OS
  • Steps đã thực hiện
